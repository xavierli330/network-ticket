工单流转平台 - 使用流程

面向运维 / 管理员 / 客户系统开发者 — 快速上手路径

管理员首次配置 (5步)

  • • ① 修改默认密码 → ② 添加工单类型
  • • ③ 添加客户 (名称/推送地址/Key/Secret)
  • • ④ 配置告警源 (Zabbix/Prometheus Webhook)
  • • ⑤ 验证: 触发测试告警，检查工单创建

告警自动流转 (无人值守)

  • • 监控系统触发 → Webhook POST → 签名校验
  • • 解析 title/severity/fingerprint → 去重检查
  • • 新工单创建 or 已有工单追加记录
  • • Worker Pool 推送客户 (⚠️ 当前待实现)

日常管理操作

  • • 工单管理: 查看/筛选/搜索/详情/重试/取消
  • • 客户管理: 新增/编辑/删除客户配置
  • • 告警源: 查看 Webhook URL，配置到监控系统
  • • 审计日志: 查看所有操作记录，支持时间筛选

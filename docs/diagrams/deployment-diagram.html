工单流转平台 - 部署架构

Docker Compose 一键部署 · 支持 HTTPS · 数据持久化

部署方式

  • • Docker Compose 一键启动 (docker-compose up -d)
  • • 5 个容器: nginx, frontend, backend, worker, postgres, redis
  • • 数据持久化: Docker Volumes (postgres_data, redis_data)
  • • 支持本地开发、测试环境、生产环境

网络配置

  • • 外部暴露: Nginx 80/443 → 内部转发到 3000/8000
  • • 内部网络: Docker bridge (容器间互通)
  • • 数据库: 仅内部访问 (5432/6379 不暴露)
  • • Webhook: 监控系统直接访问 Backend :8000

生产建议

  • • HTTPS: 使用 Let's Encrypt 或自签名证书
  • • 备份: 定时 pg_dump + volume 快照
  • • 监控: 容器健康检查 + 日志收集
  • • 扩展: Worker 可水平扩展为多实例
